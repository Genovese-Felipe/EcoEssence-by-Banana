# EcoEssence Website

This repository contains the official website for EcoEssence, a fictional brand specializing in custom natural cosmetics and sustainable packaging. The website is designed to be visually appealing, user-friendly, and informative, reflecting the brand's commitment to quality, sustainability, and personalization.

## The Brand: EcoEssence

EcoEssence was conceived to meet the growing demand for effective, sustainable, and eco-friendly beauty products. The brand's core mission is to offer high-quality, natural cosmetics that are tailored to individual needs, while also promoting environmental responsibility through innovative, reusable packaging solutions.

## Features

*   **Bilingual Content:** The website supports both English and Portuguese, with a simple language switcher.
*   **Responsive Design:** The layout is fully responsive and optimized for a seamless experience on desktops, tablets, and mobile devices.
*   **Interactive Product Configurator:** Users can create their own custom formulas through an interactive form.
*   **Detailed Product and Sustainability Information:** The website provides comprehensive details about the products and the brand's commitment to sustainability.
*   **Modern and Clean UI:** The design is modern, clean, and visually appealing, aligning with the brand's premium identity.

## Technologies Used

*   **HTML5**
*   **CSS3**
*   **JavaScript**

## Repository Structure

*   `index.html`: The main landing page.
*   `products.html`: The page showcasing the product line.
*   `sustainability.html`: A page detailing the brand's commitment to sustainability.
*   `configurator.html`: The interactive product configurator page.
*   `styles.css`: The stylesheet containing all the styles for the website.
*   `script.js`: The JavaScript file for interactive features like the language switcher and mobile menu.
