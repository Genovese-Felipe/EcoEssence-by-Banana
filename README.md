# EcoEssence by Banana

This repository contains the source code for the EcoEssence fictional brand website, a project proposed by Google Nanobanana. EcoEssence focuses on custom natural cosmetics and reusable packaging, aiming to meet the demand for effective and sustainable products that combine efficacy with environmental responsibility.

## Project Structure

The project is a static website built with HTML, CSS, and JavaScript. The main files are:

*   **`index.html`**: The main HTML file that contains the structure and content of the website. It includes sections for the hero, mission, products, packaging, values, and clients.
*   **`styles.css`**: The stylesheet that defines the visual appearance of the website. It includes styles for all sections, ensuring a consistent and responsive design.
*   **`script.js`**: The JavaScript file that adds interactivity to the website, such as the language switcher.

## Features

The website includes the following sections:

*   **Home**: A hero section with a brief introduction to the brand.
*   **Products**: A showcase of the different types of natural cosmetics offered.
*   **Packaging**: Information about the sustainable and reusable packaging.
*   **About**: The brand's mission and values.
*   **Clients**: A detailed look at the target audience and customer segments.
*   **Language Switcher**: A button to switch the website's language between English and Portuguese.
