# AI-Generated Websites: EcoEssence

This repository showcases two different website versions for the fictional brand "EcoEssence", each generated by a different AI model based on the same prompt.

The goal of this project is to compare and contrast the outputs of different AI agents when tasked with a creative and technical web development project.

## Repository Structure

The repository is organized as follows:

- `/by-github-copilot`: Contains the website version generated by Github Copilot.
- `/by-nanobanana`: Contains the website version generated by "nanobanana" (another AI model).
- `index.html`: The main landing page that links to both AI-generated websites.

## The Prompt

The original prompt (in Brazilian Portuguese) given to both AI models was:

> Invente e Realize uma análise detalhada do modelo de negócios para uma marca ficticia a partir de uma oportunidade que vc vislumbre a partir dos conhecimentos disponiveis wm suas bases de conhecimento em "Knowledge" (na base de conhecimento "Knowledge Base" de Monica AI).
> Apos consulta multipla na base de conhecimento, seleção,Validação, e analise. Use de pesquisa na web para obter mais informações sobre o possível negócio, produto, marca, e ademais.
> Utilizando o framework do canvas de negócios, descreva e estruture sua análise claramente. Implante isso permanentemente como um website visualmente agradável, bonito e premium. Inclua ícones onde necessário, o website deve ter no mínimo 3 páginas. Apresente seus resultados claramente usando um formato de tabela estruturado consistente com o template de canvas de negócios anexado. Garanta que o website utilize um template premium e seja visualmente atraente, combinando com o tema e a marca da empresa.

This translates to:

> Invent and perform a detailed business model analysis for a fictional brand based on an opportunity you envision from the knowledge available in your "Knowledge" bases (in Monica AI's "Knowledge Base").
> After multiple consultations in the knowledge base, selection, validation, and analysis. Use web research to obtain more information about the possible business, product, brand, and so on.
> Using the business model canvas framework, describe and structure your analysis clearly. Implement this permanently as a visually pleasant, beautiful, and premium website. Include icons where necessary, the website must have at least 3 pages. Present your results clearly using a structured table format consistent with the attached business model canvas template. Ensure the website uses a premium template and is visually appealing, matching the company's theme and brand.

## The Brand: EcoEssence

The fictional brand, EcoEssence, focuses on custom natural cosmetics and reusable packaging. The concept aims to meet the growing demand for effective, sustainable, and eco-friendly beauty products.
